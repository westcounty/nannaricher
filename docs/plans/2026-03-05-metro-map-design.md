# 菜根人生 —— 地铁图式地图重设计

> 设计日期: 2026-03-05
> 状态: 待实现

## 1. 设计目标

将现有 700×700 纯向量方环地图重设计为**地铁路线图风格**的商业级游戏地图，实现：

- **清晰**：玩家一眼看懂主环、支线、入口/出口关系
- **美观**：达到商业桌游级视觉品质（发光/材质/动效/深度）
- **可扩展**：每个站点预留贴图/精灵图展示空间

## 2. 当前问题

| 问题 | 原因 |
|------|------|
| 信息过载 | 700×700 画布塞了 111 个格子，8 条支线同时可见 |
| 格子太小 | 主环 70px、支线 30px，9px 字体难阅读 |
| 视觉粗糙 | 纯色块填充 + 模拟渐变，无图标/纹理/深度 |
| 支线混乱 | 直线向内延伸 + L 形出口连接线交叉 |
| 中心区浪费 | 200×200 仅放标题 |

## 3. 设计方案：地铁路线图 + 弧形支线分叉

### 3.1 整体布局

```
画布逻辑尺寸：1200 × 1000（宽屏比例）
主环形状：圆角矩形（4 角站为弯道锚点）
支线形式：从主环两个相邻站之间「弧形鼓出」
中心区域：游戏标题 + 回合信息 + 当前玩家
```

行走方向：逆时针（起点→机会→交学费→...→闯门→起点）

### 3.3 支线弧形分叉

每条支线入口和出口都是主环上的**相邻站点**。支线从入口站向外「鼓出」弧形，经过若干站后弧线回弯至出口站。

**弧形方向：**
- 底边支线（浦口、学在南哪）→ 向**下方**弧出
- 左边支线（赚在南哪、苏州）→ 向**左方**弧出
- 顶边支线（乐在南哪、仙林）→ 向**上方**弧出
- 右边支线（鼓楼、食堂）→ 向**右方**弧出

**8 条支线数据：**

|------|------|------|------|---------|--------|
| 浦口线 | 浦口线入口[4] | 紫荆站[5] | 12+1 | 下 | #607D8B 蓝灰 |

**弧线实现：** 贝塞尔曲线（3 阶），控制点向外偏移，站点均匀分布在曲线上。弧线深度（最远距离）与站点数量成正比。

### 3.4 视觉风格：「南大奇幻夜景」

#### 背景层（5 层深度）

| 层级 | 内容 | 渲染方式 |
|------|------|---------|
| 1 深空 | 极深紫黑渐变 #080515→#0F0A1A→#16102A | 径向渐变填充 |
| 2 校园剪影 | 半透明南大建筑轮廓（鼎、北大楼） | SVG 叠加 5% alpha |
| 3 发光网格 | 淡紫色网格线 + 交叉点光晕 | 1px 线条 + 圆形模糊 |
| 4 装饰边框 | 金色线条华丽中式边框 + 角装饰 | Graphics 绘制 |
| 5 大气粒子 | 缓慢浮动紫/金色雾气光点 | 粒子系统 |

#### 主环线路渲染（4 层叠加）

```
外发光层：12px 高斯模糊，#4A4A5A 30% alpha（霓虹光晕）
底衬线：8px，#2A2A3A（深色管道底）
主色线：6px，#4A4A5A → #6A6A7A 渐变（银灰金属线）
高光线：2px，白色 20%，偏上居中（金属反光）
弯道处：光晕聚集效果
```

#### 支线线路渲染

```
外发光层：8px 高斯模糊，主题色 25% alpha
底衬线：5px，主题色暗 40%
主色线：4px，主题色渐变（入口色→出口色微变）
```

### 3.5 动效系统

## 4. 技术实现要点

### 4.1 坐标系统重构

- `BoardLayout.ts` 完全重写
- 主环站点坐标：圆角矩形路径上等距分布
- 支线站点坐标：贝塞尔曲线上等距分布
- 需要实现 `getPointOnRoundedRect(t)` 和 `getPointOnBezier(t)` 工具函数

### 4.2 渲染层重构

```
新的层级结构（底→顶）：
1. BackgroundLayer   — 深空渐变 + 校园剪影 + 发光网格
2. DecorLayer        — 装饰边框 + 大气粒子
3. TrackLayer        — 主环线路 + 支线弧线（4层叠加渲染）
4. StationLayer      — 所有站点卡片（主环+支线+体验卡）
5. ConnectorLayer    — 换乘标记 + 入口/出口图标
6. PlayerLayer       — 玩家棋子 + 移动动画
7. EffectLayer       — 浮动文字 + 粒子特效 + 高亮
8. UILayer           — 中心区域信息面板
```

### 4.3 关键新增依赖

- PixiJS 粒子系统（`@pixi/particle-emitter` 或内建方案）
- 贝塞尔曲线工具（可自行实现或使用 `bezier-js`）
- 磨砂玻璃效果（PixiJS `BlurFilter` + 半透明叠加）

### 4.4 需要清理的旧代码

- 移除 `client/src/canvas/BoardRenderer.ts`（旧圆形布局）
- 移除 `client/src/canvas/colors.ts`（已被 tokens.ts 替代）
- 重构 `BoardLayer.ts`、`LineLayer.ts`、`BackgroundLayer.ts`
- 重构 `BoardLayout.ts` 坐标计算

## 5. 贴图系统设计

每个站点预留贴图位置：

```
贴图命名规范：
  主环站：/assets/stations/main/{cellId}.png  (48×48 / 80×80)
  支线站：/assets/stations/lines/{lineId}/{index}.png  (32×32)
  体验卡：/assets/stations/exp/{lineId}.png  (40×40)

Fallback：无贴图时显示 emoji 图标 或 类型色圆形占位符
加载方式：PixiJS Assets.load() 预加载，Spritesheet 打包
```
